# TetrisGame

## 👨🏿‍🏫 프로젝트 소개
Java Thread, JFrame, MVC 패턴을 적용한 TetrisGame 게임 (토이 프로젝트) 🎮
<br>점수는 파일로 저장하여 구현했습니다.

## 📝 개발 환경
* Tool: Eclipse IDE 2021-03
* Java: JDK 1.8
